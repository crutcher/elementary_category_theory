## Feedback item 1: Font size too large

**Location:** `book/main.tex` — `\documentclass[11pt,oneside]{book}`

**Issue:** The 11pt base font is too large for the page size (6×9in trim).

**Decision:** Reduce to `10pt`. This is standard for 6×9 trade paperbacks
and will bring the text density closer to a typical technical book.

If 10pt still reads large (depends on the specific font/rendering), the
next step would be `9.5pt` via the `extarticle`/`extbook` class or the
`fontsize` package — but try 10pt first.

**Status:** Pending — to be applied in next book-edit batch.

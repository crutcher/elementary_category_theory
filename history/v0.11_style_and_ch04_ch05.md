## Feedback items (directives)

### 1. Font size reduction (−2pt)

Drop the body font size by 2pt. Current was 10pt (set in v0.10);
new size is 8pt.

**Implementation note:** The standard `book` class supports only 10/11/12pt.
For 8pt we switch to `extbook` (from the `extsizes` package), which supports
8pt through 25pt natively.

### 2. Multi-line equation environments

Where several equations are presented together, or where a multi-stage
proof is being given, prefer multi-line environments (`align`, `align*`,
`gather`) over stacked `\[...\]` blocks.

For proofs with reasons, the `align* … && \text{...}` pattern is preferred:
```latex
\begin{align*}
  X &= \text{step 1}
       && \text{by rule A} \\
    &= \text{step 2}
       && \text{by rule B}
\end{align*}
```

### 3. "How to Say This" for notation

When introducing notation, include a pronunciation guide:
- **Inline form** (`\sayit{...}`) for notation with a single common reading.
- **Callout box** (`howtosay` environment) for notation with multiple
  conventional readings or where the variation is itself worth flagging.

## Style application

The two new chapters apply the v0.11 style directives:

- **Multi-line equations**: every proof with more than two steps uses
  `align*`; lists of related equations are grouped in single environments.
- **"How to say this" callouts** appear at the following notation
  introductions:
  - Ch 4: $\alpha : F \Rightarrow G$ and components; $[\mathcal{C},
    \mathcal{D}]$ / $\mathcal{D}^{\mathcal{C}}$ / $\mathrm{Fun}$;
    $F \cong G$ and natural-iso double-tilde arrows.
  - Ch 5: $\mathbf{0}$ vs. $\bot$; $\mathbf{1}$ vs. $\top$;
    products/coproducts naming triple-up.

The font size change is global via the preamble/main.tex switch to
`extbook` at 8pt.

## Pending: retrofit of existing chapters

The new style directives apply going forward. Existing chapters
(1, 2, 3, preface) were NOT retrofitted in this round:

- They contain multi-equation passages that could be combined into
  `align*` blocks (e.g., Ch 1 §1.4 unit laws, Ch 2 §2.3 inverse uniqueness
  proof, Ch 3 multi-step functoriality verification).
- They contain notation introductions (`f : A \to B`, $\mathrm{Hom}$,
  $\circ$, $\mathrm{id}_A$, $g \circ f$, $\mathcal{C}^{\mathrm{op}}$,
  $F : \mathcal{C} \to \mathcal{D}$) that could carry `howtosay` callouts.

**Decision:** Defer the retrofit. It is a substantial mechanical pass best
done as its own batch, separate from new-content writing. Logged here for
future reference.

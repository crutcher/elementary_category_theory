## Feedback item: Switch to A4 paper

**Location:** `book/preamble.tex`, `\usepackage[...]{geometry}` block

**Current setting:**
```latex
\usepackage[
  paperwidth=6in,
  paperheight=9in,
  top=0.85in,
  bottom=0.85in,
  inner=0.75in,
  outer=0.65in,
  marginparwidth=0in,
]{geometry}
```

**Decision:** Switch to A4 (210mm × 297mm). Margins will need adjustment
— the current 6×9 margins are proportioned for a small trim size. Reasonable
A4 margins for a book-style document:

```latex
\usepackage[
  a4paper,
  top=25mm,
  bottom=25mm,
  inner=30mm,
  outer=25mm,
  marginparwidth=0mm,
]{geometry}
```

The wider A4 page (vs. 6×9) will give more comfortable line lengths at 10pt.
No content changes needed — only the geometry block.

**Status:** Pending — to be applied in next book-edit batch.

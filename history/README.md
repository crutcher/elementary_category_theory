# Feedback History

This folder stores the iterative feedback record for *Elementary Category Theory*.

## Structure

Each feedback round gets a file named:

```
v<version>_<topic>.md
```

For example:
- `v0.1_initial_scaffold.md` — feedback on the first draft scaffold
- `v0.2_ch01_revision.md` — feedback on Chapter 1 revisions

## What each file contains

- **Version / date**
- **Sections covered** — which parts of the book the feedback addresses
- **Excerpts** — the actual LaTeX passages being discussed, quoted inline
- **Feedback** — specific observations, with references to the excerpt
- **Decisions** — what was changed and why, or what was deferred
- **Open questions** — things to revisit in later rounds

## Philosophy

Feedback files are permanent. We don't delete or overwrite them. They form the
record of how the book evolved and why decisions were made.
